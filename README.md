# 📊 MBA Data Analysis and Cleaning Project – Group 4

## 📌 Project Overview
This project focuses on **cleaning, preparing, and analyzing** multiple MBA-related datasets for deeper insights and possible predictive modeling.  
The datasets include application records, campaign performance, test scores, token payments, and marketing leads.

### **Goals**
- Remove inconsistencies, handle missing data, and standardize fields across datasets.
- Perform dataset-specific analysis to uncover trends and actionable insights.
- Explore relationships between applicant/test performance and marketing effectiveness.
- Prepare datasets for potential machine learning models.

---

## 📂 Datasets
We worked with five main datasets:

1. **Application Dataset** – MBA applicant details including Statement of Purpose (SOP).
2. **Campaign Performance Dataset** – Ad campaign results with cost and engagement metrics.
3. **MBA Test Dataset** – Applicant test performance records.
4. **MBA Tokens Dataset** – Token payments and related applicant details.
5. **Performance Marketing Leads Dataset** – Leads generated via marketing campaigns.
